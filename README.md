# ミライアシスト - 相談支援専門員向け必須業務支援アプリ

## 概要

ミライアシストは、障害福祉の相談支援専門員の日常業務を総合的に支援するアプリケーションです。書類作成の効率化に加えて、面談管理、期限追跡、進捗確認など、相談員の皆さんが毎日アクセスしたくなる「必須アプリ」として機能します。

## 主な機能

### ✅ 実装済み機能

#### 📊 ダッシュボード・業務管理
- **実用的ダッシュボード**: 今日やるべきタスク、進捗状況、緊急対応が必要な利用者を一目で確認
- **期限管理システム**: 制度上の期限自動計算、個人期限設定、完了追跡機能
- **面談管理**: 面談予定の登録・履歴管理、次回面談のリマインダー機能
- **カレンダー機能**: 月表示・週表示切り替え対応、面談スケジュールの可視化

#### 👥 利用者管理
- **認証システム**: PINコード認証による安全なアクセス制御
- **利用者情報管理**: 実名表示（PIN認証により安全性確保）、面談日時・場所の登録機能
- **進捗追跡**: 各利用者の書類作成状況、期限までの日数管理

#### 📄 書類作成支援
- **AI書類生成**: 4種類の書類の作成支援
  - サービス等利用計画（動的なサービス項目追加・削除機能付き）
  - 週間計画表（生成内容プレビュー機能付き）
  - ニーズ整理票（全8項目の自動生成）
  - モニタリング報告書（独立ページでの作成）
- **面談記録管理**: 音声・画像・テキストファイルのアップロード対応
- **Excel出力**: 各書類のExcelファイル出力機能
- **書類永続化**: 作成済み書類の自動保存・復元機能
- **手動作成**: AI生成に加えて手動での書類作成が全書類で利用可能

#### 🔒 セキュリティ・アクセシビリティ
- **データ暗号化**: AES256による個人情報の暗号化保存
- **アクセシビリティ対応**: 色覚多様性・スクリーンリーダー対応

### 🔄 今後実装予定

- **ローカルAI統合**: Ollama等を使用したオフライン推論
- **音声認識**: 音声ファイルの自動文字起こし
- **プロンプト管理**: 開発者向けプロンプト調整機能
- **バックアップ機能**: 自動バックアップシステム

## 技術スタック

- **フロントエンド**: React 18 + TypeScript
- **状態管理**: Zustand
- **スタイリング**: Tailwind CSS
- **データ処理**: ExcelJS（Excel出力）
- **暗号化**: crypto-js (AES256)
- **ビルドツール**: Vite
- **将来実装**: Ollama（ローカルAI推論）

## セットアップ

### 前提条件

- Node.js 18以上
- npm または yarn

1. 依存関係をインストール:
   ```bash
   npm install
   ```

2. 開発サーバーを起動:
   ```bash
   npm run dev
   ```

3. ブラウザで `http://localhost:3000` にアクセス

### ビルド

```bash
npm run build
```

## 使用方法

### 1. 初回設定

1. アプリケーションを起動
2. PINコード（4桁以上）を設定
3. 設定したPINコードでログイン

### 2. 利用者登録

1. 「利用者管理」画面から「新規利用者登録」をクリック
2. 必要な情報を入力（面談日時・場所も同時登録可能）
3. 個人情報は暗号化して保存されます

### 3. 日常業務の管理

1. **ダッシュボードで業務確認**
   - 緊急対応が必要な利用者の確認
   - 今日のタスク一覧
   - 全体的な進捗状況の把握

2. **面談管理**
   - カレンダーで面談予定を確認（月表示・週表示切り替え可能）
   - 面談予定の追加・編集・履歴管理
   - 次回面談のリマインダー

3. **期限管理**
   - 各書類の制度上期限の自動計算
   - 個人設定期限の管理
   - 期限迫る案件の自動アラート

### 4. 書類作成

1. 利用者を選択して書類作成画面へ
2. 作成したい書類の種類を選択
3. **期限設定**: 制度期限確認と個人期限設定
4. 面談記録をアップロードまたは入力
5. 「AIで書類を生成」または「手動で作成」を選択
6. 生成された内容を確認・編集
   - サービス等利用計画: サービス項目の動的追加・削除
   - 週間計画表: 生成内容のプレビュー確認
   - ニーズ整理票: 8項目の詳細な自動生成
7. 保存・Excel出力時に期限が自動的に完了となる
8. 作成済み書類は自動保存され、次回アクセス時に復元

## セキュリティ

- **データ暗号化**: すべての個人情報はAES256で暗号化
- **オフライン動作**: インターネット接続不要
- **実名表示**: PIN認証により安全性を確保した上で実名表示（業務効率向上）
- **自動ログアウト**: 8時間後に自動ログアウト
- **PIN認証**: 起動時のPINコード認証

## 対応書類

### 1. サービス等利用計画（様式2-1）
- 利用者の生活に対する意向
- 総合的な援助の方針
- 長期・短期目標
- サービス詳細（6項目まで）
- **期限管理**: 支給決定から30日以内（法定期限）

### 2. 週間計画表（様式2-2）
- 月～日の時間割
- 週単位以外のサービス
- 生活の全体像
- **期限管理**: サービス等利用計画作成後7日以内

### 3. ニーズ整理票
- インテーク情報
- アセスメント結果
- プランニング内容
- 全8項目の詳細な自動生成対応
- **期限管理**: 初回面談後14日以内（推奨）

### 4. モニタリング報告書（様式3-1）
- 援助の方針と全体状況
- モニタリング項目（6項目まで）
- 計画変更の必要性
- **期限管理**: 前回モニタリングから1ヶ月以内

## 新機能詳細

### 📅 面談管理機能
- **面談予定登録**: 利用者登録時に面談日時・場所を設定
- **カレンダー表示**: 月表示・週表示の切り替え機能
- **面談履歴管理**: 過去の面談記録を蓄積・振り返り
- **内容表示最適化**: 月表示でも内容が見切れない設計

### ⏰ 期限管理機能
- **制度期限自動計算**: 各書類の法定・制度上期限を自動算出
- **個人期限設定**: 相談員独自の期限設定機能
- **完了追跡**: 書類保存・出力時に期限を自動完了
- **ダッシュボード連携**: 期限切れ・期限迫る案件の表示
- **リマインダー機能**: 指定日数前の通知設定

### 🎯 実用的ダッシュボード
- **緊急対応リスト**: 期限切れ・期限迫る利用者の一覧
- **今日のタスク**: 本日対応が必要な業務項目
- **進捗概要**: 全体的な業務進捗状況の可視化
- **実名表示**: PIN認証による安全性確保で業務効率向上
